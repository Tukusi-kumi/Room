90年代風ブログサイトの作り方

投稿日: 2026-04-06 | カテゴリー: Web開発

こんにちは！今回はHTMLとCSSを使って、90年代風のブログサイトを作成する方法について紹介します。

なぜ90年代風デザイン？

90年代のWebデザインは、限られた技術の中で創造性を発揮した時代です。黒背景に緑文字、等幅フォント、そしてシンプルなアニメーション。これぞまさにハッカー映画に出てくるようなインターフェースです。

基本的なHTML構造

まずは基本的なHTML構造から始めましょう。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>レトロブログ</title>
</head>
<body>
    <h1>こんにちは</h1>
</body>
</html>

CSSで90年代風スタイルを適用

次にCSSでレトロな雰囲気を演出します。

背景と文字色

まずは黒背景に緑文字の基本設定：

body {
    background-color: #000000;
    color: #00cc00;
    font-family: 'Courier New', monospace;
}

発光効果の追加

text-shadowプロパティで文字を発光させます：

h1 {
    text-shadow: 0 0 10px #00cc00;
}

アニメーション効果

@keyframesを使って文字の点滅や発光のアニメーションを追加できます。

@keyframes glow {
    from {
        text-shadow: 0 0 5px #00cc00;
    }
    to {
        text-shadow: 0 0 15px #00cc00;
    }
}

.glow {
    animation: glow 2s ease-in-out infinite alternate;
}

まとめ

モダンな技術とレトロなデザインの融合は、新しい表現の可能性を広げてくれます。シンプルながらも印象的な90年代風デザイン、ぜひ試してみてください。

「シンプルであることは、複雑であることよりも難しい。」 - レオナルド・ダ・ヴィンチ
